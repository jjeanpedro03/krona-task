## 🎯 Objetivo

Desenvolver uma aplicação completa (CRUD) simulando um sistema real, com persistência de dados em nuvem e comunicação entre front-end e API.

---

## 🚀 Sobre o Projeto

Aplicação full stack com React no front-end e Node.js no back-end, com dados persistidos em banco MySQL na nuvem.

- ⚡ Integração completa entre front e back
- 🌐 Deploy em produção (Vercel + Cloud Run)
- 🧠 Estrutura simples e funcional (CRUD real)
- 🎨 Interface limpa e responsiva

## ⚙️ Funcionalidades

- Criar tarefas  
- Listar tarefas  
- Marcar como concluída  
- Deletar tarefas  
- Definir prioridade (baixa, média, alta)  

---

## 💡 Diferenciais

- 🌍 Aplicação em produção (deploy real)
- 🔗 Integração com API externa (Cloud Run)
- 💾 Banco de dados remoto (Railway)
- ⚙️ Estrutura organizada e escalável
